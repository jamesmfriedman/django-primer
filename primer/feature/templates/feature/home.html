What is Primer?

It is a framework for making web apps, with all the hard stuff done for you.

Why should I use it?

Plain and simple, building web apps should be easier. Even with so many great frameworks and tools, precious time is wasted on new projects doing redundant setup and rebuiling the same kinds of things over and over. Primer seeks to close that gap, and on day one, put you about 3 months ahead of schedule.

Built with the best.

Primer is a conglomeration of multiple open source frameworks and tools with some custom glue tying it all together. At its core, it leverages the combined powers of Django, jQuery, Bootstrap, and LESS CSS to make for the an integrated end to end development experience.

From the Author

Building web apps should be easier. I'm giving Primer away so you can spend less time worrying about how to do full site ajax page handling, and more time making fun and expressive applications.
