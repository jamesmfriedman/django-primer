What is Primer?

It is a WebDK for making web apps, with all the ground work done for you.

What's a WebDK?

Well, it's an SDK designed for the web. Primer is here to be your toolbox, with a lot of hard or boring stuff done for you.

Built with the best.

Primer is a conglomeration of multiple open source frameworks and tools with some custom glue tying it all together. At its core, it leverages the combined powers of Django, jQuery, Bootstrap, and LESS CSS to make for the an integrated end to end development experience.

From the Author

Building web apps should be quicker and easier. Primer is here to close the gap, allowing you to create high fidelity web apps with a lot of the leg work done for you.
