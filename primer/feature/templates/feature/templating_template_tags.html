{% verbatim %}{% include_raw 'mytemplate.html' %}{% endverbatim %} The template you want to include, as string. This will include a whole template without running it through Django's templating engine. This is useful if you need to include something like a set of JavaScript templates that would usually be parsed because it shares a similar syntax.
